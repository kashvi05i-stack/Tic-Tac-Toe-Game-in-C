# Tic-Tac-Toe Game in C

## Overview
This project is a simple Tic-Tac-Toe game developed using the C programming language. The game is played between a user and the computer on a 3x3 grid. The player uses 'X' while the computer uses 'O'. The program checks for winning conditions and declares the winner or a draw accordingly.

## Features
- Interactive command-line gameplay
- Player vs Computer mode
- Randomized computer moves
- Win, lose, and draw detection
- Input validation for occupied spaces

## Technologies Used
- C Programming Language
- GCC Compiler
- VS Code

## Concepts Used
- Arrays
- Functions
- Loops and Conditionals
- Random Number Generation
- Game Logic Implementation

## How to Run

### Compile
```bash
gcc tic-tac-toe.c -o tic-tac-toe
```

### Run
```bash
.\tic-tac-toe
```

## Sample Output

---|---|---
